# 回家的路－認識失智，愛不迷路

一個互動式失智症衛教遊戲，透過迷宮遊戲的形式，幫助玩家認識失智症的預防、症狀和照護知識。

## 🎮 遊戲特色

- **互動式迷宮** - 幫助郝婆婆找到回家的路
- **30道題目** - 涵蓋失智症的各種重要知識
- **知識補給站** - 每題作答後提供詳細的知識說明
- **作答紀錄** - 遊戲結束後可查看完整作答足跡
- **溫馨設計** - 公園風格的迷宮，溫暖友善的介面
- **智能追蹤** - 畫面自動跟隨郝婆婆移動，完整的走迷宮視野
- **結束遊戲功能** - 提供清楚的返回表單指引

## 🚀 在線遊玩

訪問 GitHub Pages 網址即可開始遊戲（部署後會有網址）

## 📦 本地運行

1. 下載或克隆此儲存庫
2. 直接在瀏覽器中打開 `index.html` 文件即可開始遊戲

## 🌐 部署到 GitHub Pages

### 方法一：使用 GitHub 網頁介面

1. 登入 GitHub 並創建新的儲存庫（Repository）
2. 上傳 `index.html` 文件到儲存庫
3. 進入儲存庫的 Settings > Pages
4. 在 Source 下選擇 `main` 分支
5. 點擊 Save，等待幾分鐘後即可訪問

### 方法二：使用 Git 命令行

```bash
# 創建本地 git 儲存庫
git init

# 添加文件
git add index.html README.md

# 提交
git commit -m "Initial commit: 失智症衛教遊戲"

# 連接到 GitHub 儲存庫（替換為您的儲存庫 URL）
git remote add origin https://github.com/yourusername/dementia-maze-game.git

# 推送到 GitHub
git branch -M main
git push -u origin main
```

然後在 GitHub 儲存庫設定中啟用 GitHub Pages。
